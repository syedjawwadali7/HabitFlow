# HabitFlow

HabitFlow is a modern productivity and habit tracking app designed to help users build consistency and improve daily routines. (A concept for now made for as a project but will work on it to make it better)

## Features

- Habit tracking
- Modern UI
- Dark/Light mode
- Task management
- Productivity dashboard

## Technologies

- Android Studio
- Kotlin / Java
- Firebase

## Installation

git clone https://github.com/yourusername/HabitFlow.git
